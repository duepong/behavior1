Sheet 2 / Page 2
각 상황별로 평가대상자의 도전적 행동이 얼마나 자주 보이는지 선택하세요. 0=해당사항 없음, 1=가끔씩 발생, 2=종종 발생, 3=자주 발생.
번호기능문항점수
1관심습득관심을 끌기위해 행동을 보인다.
2회피일하는 상황이나 학습상황에서 벗어나기 위해 행동을 보인다.
3자동·감각적“자기자극”의 형태로 행동을 보인다.
4신체적 상태고통을 느낄 때 행동을 보인다.
5강화물 습득좋아하는 장난감, 음식, 음료수와 같이 어떤 물건을 가지기 위해서 행동을 보인다.
6관심습득꾸중을 듣기를 즐겨하기 때문에 행동을 보인다.
7회피옷 입기, 이 닦기, 일하기 등 어떤 과제를 수행하라고 했을 때 행동을 보인다.
8자동·감각적방안에 아무도 없어도 혼자서 행동을 보인다.
9신체적 상태평가대상자가 아플 때 더 많이 행동을 보인다.
10강화물 습득평가대상자로부터 어떤 물건들을 제거하면 행동을 보인다.
11관심습득자신에게 관심을 돌리기 위해 행동을 보인다.
12회피특정 과제를 수행하기 싫어서 행동을 보인다.
13자동·감각적아무것도 할 것이 없을 때 행동을 보인다.
14신체적 상태무엇인가 자신의 신체적인 부분이 불편할 때 행동을 보인다.
15강화물 습득평가대상자가 원하는 물건을 당신이 가지고 있을 때 행동을 보인다.
16관심습득평가대상자가 당신의 반응을 보고 싶어서 행동을 보인다.
17회피다른 사람들로부터 혼자 있고 싶어서 행동을 보인다.
18자동·감각적주변의 상황을 무시하며 심하게 반복행동을 보인다.
19신체적 상태신체적으로 불편할 때 행동을 보인다.
20강화물 습득평가대상자가 원하는 물건을 또래가 가지고 있을 때 행동을 보인다.
21관심습득평가대상자가 행동을 보일 때면 “여기 와서 나를 봐” “나를 바라봐”라고 이야기하는 것 같나요?
22회피평가대상자가 행동을 보일 때면 “혼자 내버려둬” “무엇을 해라고 하지마”라고 이야기하는 것 같나요?
23자동·감각적혼자 있어도 그 행동을 즐기는 것 같나요?
24신체적 상태행동을 통해서 평가대상자가 아프다는 것을 말하는 것 같나요?
25강화물 습득행동을 보일 때 장난감, 음식, 특정 물건을 달라고 말하는 것 같나요?
